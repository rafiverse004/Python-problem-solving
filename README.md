# Python Problem Solutions

A collection of beginner-friendly Python problem solutions written with clean, readable, and well-documented code.

This repository is the companion repository to **python-practice**, where the problems are introduced and practiced. Here, every problem has a complete solution with simple explanations and clean coding practices.

## Goals

- Learn Python by solving problems
- Write clean and readable code
- Follow Python best practices (PEP 8)
- Build a strong programming foundation
- Create a useful revision resource
- Maintain a professional GitHub portfolio

## Repository Structure

```text
python-problem-solutions/
├── README.md
├── easy/
├── medium/
└── hard/
```

## Difficulty Levels

### Easy (30 Problems)

Covers:

- Variables
- Input & Output
- Operators
- Conditional Statements
- Loops
- Functions
- Lists
- Strings
- Basic Number Problems

### Medium (20 Problems)

Covers:

- Functions
- File Handling
- Dictionaries
- Lists
- Menus
- Small Projects
- Problem Solving

### Hard (10 Problems)

Covers:

- Object-Oriented Programming
- File Storage
- Larger Applications
- Game Logic
- Project Structure

## Coding Style

Each solution includes:

- A short problem description
- Clean and readable code
- Helpful comments
- Meaningful variable names
- PEP 8 formatting
- Beginner-friendly logic

The focus is on understanding the solution rather than writing the shortest possible code.

## How to Run

Clone the repository:

```bash
git clone https://github.com/<your-username>/python-problem-solutions.git
```

Navigate to the project:

```bash
cd python-problem-solutions
```

Run any solution:

```bash
python easy/01-hello_world.py
```
